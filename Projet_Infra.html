Description du Projet

Le projet Gestion de Jeux Vidéo consiste en une application web permettant de gérer une collection de jeux vidéo. Il offre des fonctionnalités complètes de gestion (CRUD) permettant d’ajouter, supprimer, mettre à jour, et visualiser des jeux via une interface utilisateur. Les données sont stockées dans une base de données MySQL hébergée sur un serveur. Le rendu de ce projet nous a aussi appris à réaliser une documentation claire et précise du projet sur git

Objectifs du Projet

    Ce projet avait pour mission de :

  • Offrir une interface permettant la gestion des jeux vidéo
  • Mettre en place une communication sécurisée entre l’application client et la base de données
  • Apprendre la manipulation et la gestion de bases de données dans un cadre concret

Stack Technique

    Le projet a utilisé les technologies suivantes :

  • Backend : PHP avec PDO pour la gestion des requêtes SQL sécurisées
  • Base de données : MySQL
  • Serveur : WAMP/XAMPP pour le développement local
  • Gestion des dépendances : Composer
  • Variables d’environnement : Dotenv pour sécuriser les informations sensibles

Sécurité et Gestion des Erreurs

    Des mesures de sécurité ont été intégrées dans le projet, incluant :

  • Prévention des injections SQL via l’utilisation de requêtes préparées PDO
  • Nettoyage et validation des données utilisateurs pour éviter toute exploitation malveillante

Ce que le Projet m’a Apporté

    Ce projet m'a permis d'apprendre et de renforcer plusieurs compétences clés :

  • Compréhension approfondie des communications client-serveur
  • >Gestion d'une base de données MySQL et des opérations CRUD
  • Maîtrise des bonnes pratiques de sécurité, comme la gestion des erreurs et la prévention des vulnérabilités
  • Première expérience dans le développement d'une application web complète

    • Ce projet fut à la fois instructif et motivant, offrant une introduction pratique à la gestion d’infrastructures informatiques.
